# Рик и Морти

Простое мобильное приложение на Flutter для просмотра и управления персонажами из мультсериала "Рик и Морти".

## Приступая к работе

Приложение загружает список персонажей из публичного API [Rick and Morty API](https://rickandmortyapi.com/documentation/), позволяет добавлять их в избранное, просматривать избранных персонажей в отдельном разделе, поддерживает кэширование данных для работы без интернета и сохраняет предпочтения пользователя локально с помощью SQLite. Интерфейс включает навигацию по нижней панели, пагинацию при прокрутке и адаптивную работу на разных устройствах.

## Установка и запуск

### Требования
- Flutter SDK версии **3.10.0 или выше**
- Dart SDK версии **3.10.0 или выше**
- IDE: Android Studio, VS Code или IntelliJ IDEA с плагином Flutter

### Шаги установки

1. **Склонируйте репозиторий:**
   ```bash
   git clone https://github.com/malko1278/rick_and_morty.git
   cd rick_and_morty

2. **Установите зависимости:**
   ```bash
   flutter pub get

3. **Сгенерируйте локализацию и код (обязательно!):**
   ```bash
   flutter clean
   flutter pub get
   flutter gen-l10n
* Убедитесь, что в pubspec.yaml есть строка flutter: generate: true. Без неё генерация локализации не сработает. 

4. **Запустите приложение:**
   ```bash
   flutter run 
* Приложение запустится на подключённом устройстве или эмуляторе (Android/iOS).
